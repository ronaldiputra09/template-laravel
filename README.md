## Technology

- Laravel 11
- Bootstrap 5.3.3 - Template Taplox

## Run Locally

Clone the project

```bash
  git clone https://github.com/ronaldiputra09/template-laravel.git
```

Go to the project directory

```bash
  cd laravel-template
```

Install dependencies

```bash
  composer update
```

Rename ENV

```bash
  .env.example -> .env
```

Generate key

```bash
  php artisan key:generate
```

Running Laravel

```bash
  php artisan serve
```
